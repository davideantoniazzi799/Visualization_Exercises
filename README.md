# Visualization_Exercises

Repository that contains personal exercises regarding several topics.

## Repository Purpose
Project goals:
- Practice data cleaning with Pandas and R;
- Produce publication-quality visualizations using Matplotlib;
- Improve exploratory data analysis skills;
- Build a portfolio of data visualization projects;

## Folder Structure
Each specific folder focuses on one of these theme. 
Each folder contains sub-folders divided into:
- *Script* folder: containing the code scripts both for the data cleaning and visualizations;
- *Data* folder: containing the original and cleaned datasets used to perform the visualizations;
- *Images* folder: containing the produced visualizations;
- *Key Findings* text file: containing analysis of the visualizations

**NOTE**: Visualization scripts have the saving image command commented. To save the charts in your device, uncomment those code lines.

## Programming Languages
Python and R
